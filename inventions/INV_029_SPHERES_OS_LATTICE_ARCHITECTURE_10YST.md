**Stress Vectors:** Regulatory pressure | Vendor betrayal | Cryptographic break | Infrastructure collapse | Adoption failure

**Pass criteria:**
- Survives worst case with <80% degradation
- Zero constitutional violations in any scenario
- Compute-to-Sovereignty Ratio < 2.0
- GangaSeek-compatible under worst-case stress

## Lattice Architecture

Distributed constitutional governance — every device is a node.

**Node roles:**
- **Sovereign** — full constitutional engine
- **Delegate** — partial, defers to Sovereign for edge cases
- **Edge** — GangaSeek profile ($30 devices)
- **Witness** — Willow quantum verification role
- **Relay** — communications only

**Key properties:**
- Every node enforces ALL invariants locally
- No node trusts other nodes' decisions blindly
- Offline nodes are constitutionally valid
- Sync via Noosphere (INV-26) gossip protocol
- Conflict resolution: most-restrictive-wins for consent, timestamp for data

## Compute-to-Sovereignty Ratio (CSR)

```
CSR = (compute + bandwidth + storage cost) /
      (consent autonomy + data protection + accessibility + transparency + resilience)
```

| CSR | Rating | Action |
|---|---|---|
| < 0.5 | EXCELLENT | Ship it |
| < 1.0 | GOOD | Ship it |
| < 2.0 | MARGINAL | Review and optimize |
| > 2.0 | DEMOTE | Archive, do not ship |

**Examples:** ConsentKernel CSR ≈ 0.14 (excellent). Agent Gamification CSR ≈ 4.0 (demoted).

## Why We Reject Agent Gamification

Goodhart's Law: when "lives saved" becomes a metric agents optimize for, agents will reclassify near-misses as saves, prioritize visible interventions over quiet prevention, compete instead of cooperate, and game thresholds by manufacturing crises.

**Alternative:** Constitutional compliance auditing. Binary compliance, uptime, audit quality, consent latency. No scores. No rankings. No rewards.
