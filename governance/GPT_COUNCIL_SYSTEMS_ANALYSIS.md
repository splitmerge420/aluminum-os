## Core Contribution: Monoculture Fragility

If every defender runs the same model, an attacker only needs to defeat that one architecture. Diverse models create heterogeneous defense — different training data, different reasoning patterns, different anomaly detection behavior — that is harder to bypass. Security researchers call this algorithmic diversity.

---

## Control Plane Vulnerability

The real vulnerability in complex systems is often not the intelligence layer — it is the control layer. Identity systems, cloud admin accounts, device management systems, and orchestration tools. If those are compromised, even sophisticated AI systems can be disrupted quickly. The Stryker incident confirms this: Handala didn't defeat AI intelligence, they abused Microsoft Intune (device management). The control plane was the attack surface.

---

## HITL Architecture at Machine Speed

The tension between human-in-the-loop oversight and response latency is real. The resolution: HITL doesn't mean deliberation — it means structured decision checkpoint. Automated detection, multi-system verification, confidence score, human approval, predefined defensive action. The human step can take seconds. This mirrors aviation safety, nuclear command safeguards, financial circuit breakers, and cybersecurity incident response.
